## Instrucciones para el Logo

Por favor, coloca tu logo de la empresa en esta ubicación con el nombre:
`irs-logo-chico.png`

**Ubicación**: `src/assets/img/irs-logo-chico.png`

### Especificaciones Recomendadas:

- Formato: PNG (con transparencia)
- Tamaño recomendado: 150x150 píxeles o similar
- Peso máximo: 200KB

Si no tienes un logo, puedes crear uno temporalmente o usar un placeholder.
